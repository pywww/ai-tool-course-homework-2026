### 第二讲：Git与Gitee基础

#### 作业提交流程（重要！）
1. 创建个人分支：`git checkout -b homework/姓名_学号`
2. 完成作业后，先同步最新内容：`git checkout master` → `git pull`
3. 切换回个人分支并合并：`git checkout homework/姓名_学号` → `git merge master`
4. 解决冲突（如有）
5. 推送到Gitee：`git push origin homework/姓名_学号`
6. 创建Pull Request

#### Gitee Pages个人网页创建
1. 创建仓库：`你的用户名.gitee.io`
2. 创建 `index.html` 文件
3. 在"服务"中开启Gitee Pages
4. 访问：`https://你的用户名.gitee.io`
